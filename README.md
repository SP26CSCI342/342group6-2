This project is a online arcade, the arcade allows users to play classic arcade games, and compare their highscores to those of other users    
    
Setup:    
clone repository
in terminal, run    
npm install    
    
how to run:    
in terminal, run    
npm run dev    
user can now interact with the frontend of the project from the localhost link output by the terminal, usually    
http://localhost:5173    
    
Navigation:    
navigation from the main page is done via the links under the main page leaderboard, and the login and signup buttons   

currently only the asteroids page works, clicking on the other links sends the user to the skeleton for the page not found page, and the login and signup buttons send the user to blank pages        

The asteroids page allows the user to play a version of the game asteroids, the play button launches the game, the profile page brings up the users game profile (currently empty), and the leaderboard button launches the leaderbaord for the game which at this moment uses mock data.
